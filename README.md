# 法律风险初筛工具

这是一个面向中国大陆使用场景的静态网页项目，别人拿到公开链接后，可以直接在手机浏览器里打开使用。

## 现在已经具备的能力

- 六类常见法律场景初筛
- 根据场景动态切换“关键触发点”
- 生成风险等级、提示和可复制摘要
- 页面内置“分享页面”和“复制页面链接”按钮
- 适配手机浏览器显示
- 支持 GitHub Pages 直接从分支发布

## 怎样分享给别人看

要让别人也能在手机上看到，最简单的方法就是部署到 GitHub Pages。

### 方式一：GitHub Pages

1. 把当前目录推送到你的 GitHub 仓库。
2. 进入仓库 `Settings > Pages`。
3. 在 `Build and deployment` 里把 `Source` 改成 `Deploy from a branch`。
4. 分支选择 `master`，目录选择 `/ (root)`，然后点 `Save`。
5. 等 1 到 3 分钟，GitHub 会自动发布页面。
6. 把生成的网址发给别人，对方手机点开就能使用。

部署成功后的地址通常类似：

```text
https://你的用户名.github.io/你的仓库名/
```

### 方式二：直接发本地文件

不推荐。因为 `index.html` 本地文件只能你自己打开，别人手机上无法直接访问你电脑里的文件。

## 页面里的分享按钮怎么用

- 如果网页已经部署到线上：
  - 点击“分享页面”会优先调用手机系统分享面板
  - 点击“复制页面链接”会直接复制当前网页地址
- 如果网页还只是本地打开：
  - 页面会提示你先部署，再分享
