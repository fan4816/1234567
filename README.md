# 法律风险初筛工具

这是一个面向中国大陆使用场景的静态网页项目，别人拿到公开链接后，可以直接在手机浏览器里打开使用。

## 现在已经具备的能力

- 六类常见法律场景初筛
- 根据场景动态切换“关键触发点”
- 生成风险等级、提示和可复制摘要
- 页面内置“分享页面”和“复制页面链接”按钮
- 适配手机浏览器显示
- 自带 GitHub Pages 自动部署工作流

## 怎样分享给别人看

要让别人也能在手机上看到，最简单的方法就是部署到 GitHub Pages。

### 方式一：GitHub Pages

1. 把当前目录推送到你的 GitHub 仓库。
2. 仓库开启 GitHub Pages。
3. 代码推送后，`.github/workflows/deploy-pages.yml` 会自动部署。
4. 部署完成后，你会得到一个公开网址。
5. 把这个网址发给别人，对方手机点开就能使用。

部署成功后的地址通常类似：

```text
https://你的用户名.github.io/你的仓库名/
```

### 方式二：直接发本地文件

不推荐。因为 `index.html` 本地文件只能你自己打开，别人手机上无法直接访问你电脑里的文件。

## 页面里的分享按钮怎么用

- 如果网页已经部署到线上：
  - 点击“分享页面”会优先调用手机系统分享面板
  - 点击“复制页面链接”会直接复制当前网页地址
- 如果网页还只是本地打开：
  - 页面会提示你先部署，再分享

## 我已经为你准备好的部署文件

- 页面主文件：`index.html`
- 自动部署工作流：`.github/workflows/deploy-pages.yml`

## 你下一步只差什么

当前目录已经是本地 Git 仓库，但还没有接上你的 GitHub 远程仓库，所以现在还不能直接生成公开链接。

如果你把下面两项中的任意一项给我，我就可以继续帮你完成：

1. 你的 GitHub 仓库地址
2. 你的 GitHub 用户名和准备使用的仓库名

到那一步，我可以继续帮你：

- 连接远程仓库
- 提交当前代码
- 推送到 GitHub
- 告诉你最终可分享的手机访问链接
